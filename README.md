# travel

> 来自慕课网去哪儿网APP部分页面渲染小项目 

## Build Setup

``` bash
# 安装依赖项
npm install

# 开发环境下
npm run dev

# 项目打包
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```

## 技术栈

> 项目虽小但是该用上的技术还是用到了

```bash
Vue
Vuex
Vue-router
Vue-cli
axios
mock
```

学习Vue后不知道怎么统一运用这个强大的框架，再接触这个项目后大致的还是了解下整改项目的开发流程。总的来说收获还是有的
